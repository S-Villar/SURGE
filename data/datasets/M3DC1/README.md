# SPARC M3DC1 (optional)

ROSE examples can use ``--dataset m3dc1`` when a pickle is present.

1. Copy the bundle from your SURGE tree, e.g.::

       cp /path/to/SURGE/data/datasets/SPARC/sparc-m3dc1-D1.pkl \
          ../../data/datasets/M3DC1/sparc-m3dc1-D1.pkl

2. Keep ``sparc_m3dc1_D1_metadata.yaml`` committed (inputs/outputs for SURGE).

3. Default demos use ``--dataset synthetic`` and need no PKL.
