# SURGE logos

This directory holds the design-system logos used by the README and the
Sphinx docs. Two variants, both on the dark navy background:

* `surge_logo_os.png` — square mark (1254 × 1254). Used as the inline icon
  next to the "SURGE" title in `README.md`.
* `surge_logo_os_expanded.png` — wide wordmark (1672 × 941). Used as the
  banner in `README.md` and as the hero image in `docs/index.rst`.

If you add new logo variants, keep the naming `surge_logo_*.png` and
drop them here — the `.gitignore` has an explicit allow-list for
`data/logos/*.png`.
