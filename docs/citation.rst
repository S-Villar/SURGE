Citing SURGE
============

The software is registered in `U.S. DOE OSTI / DOE Code
<https://www.osti.gov/doecode>`__ (Code ID `179819
<https://www.osti.gov/doecode/biblio/179819>`__) with DOI
`10.11578/dc.20260422.5 <https://doi.org/10.11578/dc.20260422.5>`__.
OSTI’s project page also exports APA, Chicago, **MLA**, BibTeX, EndNote,
and RIS under *Citation Formats*. The auto-generated MLA there can
**repeat** names; use the line below (three authors, no duplicates)
for papers and bibliographies.

**MLA (recommended):**

Sanchez-Villar, Alvaro, Churchill, R. Michael, and Jha, Shantenu. *SURGE - Surrogate Unified Robust Generation Engine.* Computer Software. `https://github.com/S-Villar/SURGE <https://github.com/S-Villar/SURGE>`__. USDOE Office of Science (SC), Fusion Energy Sciences (FES). 22 Apr. 2026. Web. doi:10.11578/dc.20260422.5.

The repository’s root ``CITATION.cff`` lists the same three people once
each and matches this author list.

The ``CITATION.cff`` also carries the DOI, the OSTI page URL, and
``preferred-citation`` for tooling. Version and URL should match the
release you used (see ``pyproject.toml`` and tags on
`GitHub <https://github.com/S-Villar/SURGE>`__).
